Welcome to QuadChat

Query four AI models in parallel and compare their responses side by side.

Let's get you set up in 5 quick steps.

Add Your API Keys

Get keys from each provider:

OpenAI Anthropic Google xAI

Enter keys in Settings (gear icon).

Add Context

Give all models the same instructions and documents.

System Prompt: Set behavior like "You are a coding assistant"
Documents: Upload PDFs or text for shared context

Use the sidebar to add these.

Choose Your Models

Click any provider logo in the chat header:

Toggle: Enable or disable providers
Model: Pick version (GPT-5, Claude Opus, etc.)

Active providers appear in the response grid.

Install Desktop App

Launch QuadChat from your Desktop.

Optional - you can always use the browser.

You're Ready!

Start chatting and compare how different models respond.
